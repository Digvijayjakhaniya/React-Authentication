<h1>React Authentication</h1>
  <p><p>This project is a static website built using React, designed to showcase various user interactions typically found in dynamic web applications. The site includes a home page, contact page, and user authentication features such as registration, login, password reset, and password change. It also features a user dashboard to display personalized content.  the site incorporates front-end validations to ensure a smooth user experience.</p>
</p>

  <h2>Features</h2>
  <ul>
    <li><strong>Home Page:</strong> Description of what users can find on the home page.</li>
    <li><strong>Contact Page:</strong> How users can reach out through the contact page.</li>
    <li><strong>User Registration and Login:</strong> Steps for users to register and log in, including validation details.</li>
    <li><strong>Reset Password:</strong> Instructions on how users can reset their password if forgotten.</li>
    <li><strong>Change Password:</strong> How users can change their password.</li>
    <li><strong>Dashboard:</strong> Overview of what users can do on the dashboard page.</li>
  </ul>

  <h2>Installation</h2>
  <ol>
    <li><strong>Clone the repository:</strong></li>
    <pre><code>git clone https://github.com/Digvijayjakhaniya/React-Authentication.git
cd React-Authentication
</code></pre>
    <li><strong>Install dependencies:</strong></li>
    <pre><code>npm install
</code></pre>
  </ol>

  <h2>Usage</h2>
  <ol>
    <li><strong>Run the development server:</strong></li>
    <pre><code>npm start
</code></pre>
    <li><strong>Open in your browser:</strong></li>
    <pre><code>http://localhost:3000
</code></pre>
  </ol>

  <h2>Technologies Used</h2>
  <ul>
    <li>React</li>
    <li>MUI: The React component library</li>
  </ul>
